
# 财务报表自动分类清洗工具 (V2.0)

这是一个基于 **Python + PyQt5 + Pandas** 开发的财务数据处理工具。主要用于解决从财务软件导出的旧版 `.xls` 报表存在乱码、格式陈旧、物料分类杂乱等问题。

## 🌟 核心功能

- **无痕格式转换**：自动识别 `.xls` 文件编码并修复“锟斤拷”等乱码，隐式转换为 `.xlsx` 处理，并在使用后自动删除临时文件，保持文件夹整洁。
- **自动化清洗**：
    - 自动剔除报表底部的“合计”行。
    - 自动提取“科目编码”、“科目名称”、“本期贷方发生数量”关键列。
    - 自动过滤所有数值为 0 的无效数据。
- **SQLite 字典驱动**：使用本地 SQLite 数据库 (`dictionary.db`) 存储分类规则，支持数万条数据的高速检索。
- **智能学习归类**：当遇到字典中未记录的新科目时，程序会弹出交互窗口引导用户归类，归类结果将自动永久同步至数据库。
- **分表分Sheet导出**：
    - **Sheet1 (原材料及返工旧药)**：自动合并原材料与返工类产品。
    - **Sheet2 (包材)**：独立存放包装材料类数据。
- **全方位字典管理**：支持从 Excel 导入现有字典、备份数据库为 Excel、以及在程序内直接进行增删改查。

## 📂 项目结构

```text
project/
├── main.py          # 程序入口，负责界面展示与业务调度
├── converter.py     # 核心转换模块，处理 XLS 转码与乱码修复
├── dictionary.py    # 数据持久化模块，负责 SQLite 数据库操作
├── dialogs.py       # UI 弹窗模块，处理人工分类与字典编辑界面
└── dictionary.db    # (自动生成) 存储分类规则的本地数据库
```

## 🛠️ 环境要求

- **Python 版本**: 3.7+
- **依赖库**:
  ```bash
  pip install PyQt5 pandas xlrd openpyxl
  ```
  *注：`xlrd` 版本需支持旧版 xls 读取。*

## 🚀 使用指南

1. **启动程序**：
   运行 `python main.py`。
2. **初始化字典**：
   - 首次使用，可以通过“字典管理 -> 下载字典标准模板”填写后，再通过“上传 Excel 导入数据库”一次性导入现有规则。
   - 也可以在处理报表过程中通过弹窗逐步建立。
3. **处理报表**：
   - 点击“上传财务报表 (.xls) 并处理”。
   - 如果报表中存在未知科目，请根据弹窗提示指派分类（原材料/包材/返工旧药）。
4. **检查与导出**：
   - 在主界面 Tab 页中预览分类后的结果。
   - 点击“导出为分表 Excel”，选择保存位置，即可获得按 Sheet 分类好的结果。

## ⚠️ 注意事项

- **分类名称**：程序内部严格逻辑匹配 **“原材料”**、**“包材”**、**“返工旧药”** 这三个关键词。在手动编辑字典时，请确保文字准确。
- **数据备份**：建议定期通过菜单栏的“备份数据库”功能导出 Excel，以防本地 `.db` 文件损坏导致规则丢失。
- **文件占用**：处理或导入字典时，请确保目标 Excel 文件未被 Office 等软件占用。
